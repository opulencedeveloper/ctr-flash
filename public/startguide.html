Quick start guide on CTR Flash

This guide will help you set up and start using CTR Flash efficiently, covering key features and best practices.

Table of contents

How to create a project

To add a new project, first navigate to the "Project" tab on the platform. Once you're there, find and click the "Add Project" button to begin setting up your new project.

You can add as much project as you want

Enter a unique name for your project and provide a detailed description to clarify its purpose. Once you've customized these fields, simply click the "Create Project" button to finalize and set up your project.

Finally, your project is created successfully

How to create campaigns

Find the "Campaign" tab on your dashboard, then click the "Create Campaign" button to begin setting up your new campaign.

You can add as much project as you want

Select your desired campaign type, then fill in the title and website information. Once completed, click the "Create Campaign" button to finalize your setup.

Finally, proceed to set up your campign

How to add keywords

For each individual campaign, locate and click on the "Add Keywords" button to include relevant keywords for that specific campaign.

You can add as much project as you want

Next, move forward to set up your keywords in this section. You can add as many keywords as you want to optimize your campaign.

You can view the performance of each individual keyword by clicking on the three dots next to the keyword and then selecting “View.”
